## Установка
   
1. Перейти в директорию проекта:

    ```bash
    cd ваш_проект
    ```

2. Cоздайте виртуальное окружение

    ```bash
    python3 -m venv venv
    ```

3. Активируйте виртуальное окружение

    - Unix

    ```bash
    source venv/bin/activate
    ```

    - Windows

    ```bash
    venv\Scripts\activate
    ```

4. Установить зависимости:

    ```bash
    pip install -r requirements.txt
    ```

## Использование

```bash
    python game.py
```
